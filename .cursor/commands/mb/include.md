# Include Memory Bank

When this command is invoked, you MUST:

1. Read ALL files in the `memory-bank/` directory (as defined in `.cursor/rules/memory-bank.mdc`)

2. Apply the rules from:

   - `.cursor/rules/core.mdc` (Plan/Act mode rules)
   - `.cursor/rules/memory-bank.mdc` (Memory Bank workflow rules)

3. Follow the Memory Bank workflows for the remainder of this session:
   - Update documentation when making significant changes
   - Follow Plan/Act mode as defined in core.mdc

This command activates the full Memory Bank system for complex tasks that require project context and documentation workflows.
